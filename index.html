Почему Max, а не Pro?

Claude Max даёт значительно больше лимитов на Opus 4.6 — самую мощную модель. В CLI-версии это критично: агент может работать автономно часами, и лимиты Pro быстро заканчиваются. Для первой встречи Pro хватит, но на боевое использование бери Max.

Главный инструмент — терминальный AI-агент

Это не просто чат. Claude Code — автономный агент: читает файлы, понимает контекст, пишет и запускает код, управляет Git, деплоит на серверы, подключается через MCP.

Вариант A — Terminal (рекомендуется)

Terminal

⚠️ Node.js больше не нужен

Нативный установщик работает без зависимостей. macOS 13+ и 4 GB RAM.

Что такое MCP

Model Context Protocol — «разъёмы» для Claude

MCP — способ подключить Claude к внешним сервисам. Вместо того чтобы копировать данные руками, Claude сам заходит в ClickUp, Telegram, Google Sheets, браузер и работает с ними напрямую.

📋
Пример с ClickUp

«Посмотри мои задачи на эту неделю и предложи приоритизацию». Claude через MCP подключается к ClickUp API, вытягивает задачи, анализирует и предлагает порядок.

ClickUp MCP Setup
# Подключить ClickUp MCP: claude mcp add --transport http --scope user \ clickup https://mcp.clickup.com/mcp   # Подключить Circleback MCP: claude mcp add --transport http \ circleback https://app.circleback.ai/api/mcp   # Авторизация обоих через OAuth: /mcp

Настрой CLAUDE.md

«Конституция» для твоего AI-агента

Это файл-инструкция, который Claude читает при каждом запуске. Туда записываются правила, стиль работы, контекст проекта. На встрече мы создадим твой первый CLAUDE.md.

Создать CLAUDE.md
# Создай файл в корне проекта: touch CLAUDE.md   # Что писать внутри: # - Чем занимается компания # - Какие инструменты используете # - Стиль коммуникации и тон # - Правила и ограничения для агента
